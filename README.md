## Setup
1. `python -m venv .venv`
2. `.venv\Scripts\activate`
3. `pip install -r requirements.txt`
4. `python -m ipykernel install --user --name=TfluidosT1 --display-name "Python (TfluidosT1)"`
5. Abrir `turbulence_analysis.ipynb` e no canto superior direito selecionar **Python (TfluidosT1)** no kernel.
